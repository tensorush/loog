## :lizard: :sweat_drops: **loog**

[![CI][ci-shd]][ci-url]
[![LC][lc-shd]][lc-url]

### [Server log](https://en.wikipedia.org/wiki/Common_Log_Format) analyzer.

#### :rocket: Usage

```sh
git clone https://github.com/tensorush/loog.git
cd loog/
zig build exe -- <input.log> <output.json>
```
